# Roadmap

## Slice 1: Repository Spine

Create the Turborepo workspace, shared TypeScript configuration, documentation, Docker services, GitHub Actions, initial Prisma schema, and first domain tests.

## Slice 2: Accounts and Catalogue

Add Auth.js account flows, product catalogue browsing, search foundations, and admin catalogue management backed by Prisma.

## Slice 3: Inventory and Suppliers

Implement stock tracking, reservations, supplier records, supplier SKUs, purchase metadata, reorder thresholds, and admin receiving workflows.

## Slice 4: Commerce and Shipping

Implement cart, Stripe checkout, order capture, UK shipping, worldwide shipping, tax-ready order totals, fulfilment states, and customer order history.

## Slice 5: Hobby Workflows

Implement wishlist, deck builder, collection manager without value tracking, buylist intake, tournament registration, and rewards ledger.

## Slice 6: CMS and Admin Dashboard

Add CMS pages, merchandising blocks, admin dashboards, operational reports, and audit-friendly staff workflows.

## Slice 7: Mobile Companion

Deliver Expo customer workflows for account access, collection management, wishlist, catalogue search, tournament registrations, and rewards visibility.
