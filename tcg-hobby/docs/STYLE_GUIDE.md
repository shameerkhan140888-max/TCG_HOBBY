# Style Guide

## Brand Feel

TCG Hobby uses a premium dark theme with a confident orange accent. The product should feel commercial, trustworthy, fast, and focused on collectors and players.

## UI Principles

- Prefer dense, clear retail workflows over marketing-heavy layouts.
- Use orange for primary action, focus, and selected states.
- Use neutral dark surfaces with strong contrast.
- Keep cards restrained and purposeful.
- Use icons for compact actions where the meaning is familiar.
- Avoid decorative clutter that competes with product imagery.

## Content Principles

- Write concise retail copy.
- Use direct labels for actions.
- Avoid exaggerated collector value claims.
- Never imply collection value tracking exists.
- Be explicit about shipping destinations, tournament details, and buylist terms.

## Code Style

- TypeScript strict mode.
- Domain-first naming.
- Framework code stays at application boundaries.
- Tests should cover business rules, API contracts, and important user flows.
- Documentation changes accompany architecture or behaviour changes.
