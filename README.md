# TCG Hobby

TCG Hobby is a production-ready trading card game commerce platform for Project Atlas. It is designed for premium retail experiences across web, API, admin, and mobile channels.

## Product Scope

- Customer accounts
- Wishlist
- Deck Builder
- Collection Manager without value tracking
- Buylist
- Tournament registration
- Rewards
- Product catalogue
- Search
- Live pricing where permitted
- Supplier management
- Inventory management
- Admin dashboard
- CMS
- UK and worldwide shipping

## Technology Direction

- Turborepo monorepo
- Next.js 15 storefront
- NestJS API
- React Native Expo mobile app
- Admin portal
- PostgreSQL
- Prisma
- Auth.js
- Stripe
- Tailwind CSS
- shadcn/ui
- TypeScript
- Docker
- GitHub Actions

## Engineering Principle

The repository is built incrementally. Every meaningful change should compile, include relevant tests, and update documentation when behaviour or architecture changes.
